한 줄 정의

꿈을 텍스트 또는 음성(마이크)으로 말하면, 스킬·에이전트 지식을 품은 Claude가 대화형으로 해몽해주는 PWA. 후속 질문으로 해석이 깊어지고, 응답은 TTS로 읽어준다. 회원가입 없음, 서버 DB 없음, 브라우저 로컬에 세션 100건 저장.
※ Phase 3 재설계: 채팅 UI + 음성 I/O + 안전 분류기 제거 + 캐주얼 고정 + 스킬 품질 전면 통합
※ relay 전환 + 모델 선택(2026-07-08): LLM 호출을 gugbab-claude-relay 프록시 경유 Claude로 전환, 사용자가 모델 4종(haiku·sonnet·opus·fable) 중 직접 선택
※ 줌 차단(2026-07-15): 핀치/더블탭 확대로 UI 깨지는 문제 방지 — viewport maximumScale=1·userScalable=false·viewportFit=cover + body touch-action: manipulation (gugbab-health와 동일 패턴)
※ 해몽 톤 전환(2026-07-16): deep 프롬프트를 관점(전통·심리학·과학) 나열식 → 꿈해몽가와 마주 앉아 이야기하는 대화체로 전환. 관점 표찰·소제목·불릿 금지, 지식은 흐름에 녹임. 안전 규칙(단정 예언·진단 금지) 유지
